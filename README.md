## 📖 關於專案

本專案將擁有數千年歷史的卜卦數位化，採用最為複雜的「大衍筮法」演算法進行虛擬起卦，並嚴格遵循南宋理學家朱熹的解卦法則來提取關鍵爻辭。


## ✨ 核心功能

*   **☯️ 大衍筮法**：還原傳統49根蓍草的機率分佈，使每次起卦皆與天地共振。
*   **📜 雙語資料庫**：內建64卦、384爻的完整中英雙語JSON資料庫，包含卦名、卦辭與精準翻譯。
*   **💡 朱熹解卦**：系統會自動判斷0到6個動爻的變化，精準提示使用者該閱讀哪一段卦辭或爻辭。
*   **🌐 雙語切換**：一鍵切換繁體中文與全英文介面，排版精美，適應各種螢幕尺寸。
*   **🤖 Prompt生成**：動態生成結合占卜狀態的專屬Prompt，讓AI成為你的私人解卦師。
